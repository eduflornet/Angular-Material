# Angular-Material
Styling Application with Angular Material

## Used tools:

- Angular CLI: 11.2.1
- Angular Material: 11.2.1
- Angular Flex Layout: 11.0.0-beta.33  (FlexBox CSS + mediaQuery)
- Typescript: 4.1.2
- NodeJS: 14.15.4
